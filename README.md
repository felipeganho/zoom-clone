Instruções:

  
```bash
# Clone este repositório
$ git clone git@github.com:felipeganho/zoom-clone.git

# Acesse a pasta do projeto 
$ cd zoom-clone

# Abra o projeto com o VSCode
$ .code 

# Abra cada pasta (como feito abaixo) em rode:
$ npm ci --silent

# Abra a pasta /public em um terminal e rode:
$ npm start

# Abra a pasta /server em um terminal e rode:
$ npm start

# Abra a pasta /peer-server em um terminal e rode:
$ npm run dev
